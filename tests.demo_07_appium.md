**tests.demo_07_appium**

0. Prerequisites

    [Node.js](https://nodejs.org/en/)
    Notes: Preferably LTS releases.
    
1. Install

```
    npm install -g appium
```

Maven dependency: 

```
        <dependency>
            <groupId>io.appium</groupId>
            <artifactId>java-client</artifactId>
            <version>4.1.2</version>
        </dependency>
```
